Making Musical Code workshop

Have you ever used code to generate a melody? By the end of this workshop, each participant will map input text to musical notes and then generate a melody using Python. Good-to-haves include headphones, a will to use Python libraries, arrays, and more, and a will to explore basic musical concepts.


Katarina Hoeger

Steam Instructor, The Digital Arts Experience, @kfhoeger

Katarina is currently a STEAM instructor and musician. As a musician or sound artist, she integrates mathematics and coding into some of the sound experiences she create.
